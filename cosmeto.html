Les ateliers


Fabriquez votre baume à lèvres


Les lèvres sont particulièrement sensibles au desséchement causé par les fortes chaleurs ou encore par le froid
Il est important d’en prendre soin
Pourquoi ne pas fabriquer votre propre baume à lèvres avec nous ?
Tous les lundis 15h-17h : 20€



Fabriquez votre gommage corporel


Les gommages permettent d’éliminer les cellules mortes accumulées à la surface de l’épiderme
Après un gommage, la peau absorbe mieux les actifs hydratants et nourrissants contenus dans les soins
Nos formateurs vous aideront à réaliser le votre, il deviendra le produit start de votre salle de bain !
Tous les mardis 15h-17h : 20€



Fabriquez votre bombe effervesecente pour le bain


Venez fabriquer une délicieuse bombe de bain
Regardez-la se disperser dans votre bain
Respirez ces huiles essentielles de bergamote, d'ylang, ou encore de jasmin
Savourez cet instant magique depuis votre salle de bain
Décontractez vous
Nos ateliers vous donneront les bases pour créer la bathbomb adaptée à vos besoins !
Tous les samedis 15h-17h : 20€
